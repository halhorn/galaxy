# Story 4: 物体編集

## 目的

物体の追加・削除・位置変更をインタラクティブにできるようにする。

## 計画概要

### Phase 1: オブジェクト選択・情報表示
レイキャストで物体をクリック選択、情報 (質量・速度・位置) を表示。

### Phase 2: 追加・削除・移動
UI から物体を追加/削除、ドラッグで位置を変更。質量・初速の設定。

## 受け入れ条件

- [ ] 物体をクリックして選択できる
- [ ] 選択した物体の情報 (質量・速度・位置) が表示される
- [ ] 新しい物体を追加できる (質量・位置・初速を指定)
- [ ] 物体を削除できる
- [ ] ドラッグで物体の位置を変更できる
